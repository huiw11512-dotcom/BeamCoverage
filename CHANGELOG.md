# BeamCoverage 更新说明

## 3.1.1

- 将当前三维包络、三维方向图和扫描并集三维预览的交互显示层切换为 PyQtGraph OpenGL；计算核心和 MATLAB 原型复现逻辑不变。
- 三维图拖拽、缩放、平移时不再依赖 Matplotlib 3D 逐帧重绘整张坐标轴，降低“卡顿”和文字重叠风险。
- OpenGL 不可用或自动化无界面测试环境下自动回退到 Matplotlib 渲染，保证打包 smoke test、Excel 报告和离线验证流程仍可运行。
- 当前 PNG 导出兼容 OpenGL 视图截图，普通二维图仍使用原 Matplotlib 导出路径。
- 同步 Windows EXE 元数据、软件内版本号、GitHub 更新检查版本和 Android Preview versionName。

## 3.1.0

- 默认启用自动采样策略：根据频率、阵列电尺寸、单元间距/波长比、阵列长宽比和计算模式自动提高二维切面、三维包络、u-v 方向图、扫描并集和 3D 显示网格采样。
- 保留“自定义计算/显示采样”作为高级调试入口，但常规使用不再需要手动反复调整采样数和扫描步进。
- 参数编辑与图形计算解耦：修改左侧参数后只更新推导结果和“待计算”状态，右侧图形保留上一次计算结果，点击“计算”后才刷新。
- 关闭软件时自动保存上次参数、计算项勾选、窗口尺寸、分隔栏和当前图页；下次启动自动恢复。
- 新增软件内“更新说明”页，记录从原型到当前版本的主要修复、功能和兼容性变化。
- 新增 GitHub 更新检查入口：可从“帮助 > 检查更新”访问 GitHub Releases，启动时每天最多轻量检查一次；无网络时不影响离线计算。

## 3.0.12

- 支持矩形、圆形/椭圆、菱形阵列排布和矩形、圆形/椭圆、菱形单元口径，并统一口径面积、远场模型和重叠校验入口。
- 支持导入阵元坐标 CSV，导入阵元和参数生成阵元共用同一套几何校验、功率统计和绘图流程。
- 单元远场方向图导入支持 CSV/TXT/DAT 表格和 CST/HFSS 常见 FFD/FFS 矢量远场网格。
- 单元近场接口支持 CSV/TXT/DAT 表格校验，并提供近场投影生成远场方向图 CSV 的实验功能。
- 修复窗口标题、图标、EXE 元数据、发布目录、ZIP、Excel 和 HTML 文件名中的旧原型命名残留，统一为 BeamCoverage。
- 增加暴力测试：随机核心计算、几何合法性、导入阵列、导入单元方向图、导出和打包后 EXE smoke test。

## 3.0

- 增加三维方向图页，区分“方向图 Pattern(dB)”和“S=S0 等 EIRP/功率密度包络”。
- 扫描并集页增加固定 x-z/y-z 切面包络和三维预览，避免只看三维面时误判。
- 改善三维表面显示：连续网格面替代散点显示，增加坐标轴、刻度和色标。
- 改善二维切面：固定 x-z 和 y-z 切面，不随主瓣旋转；补齐尺寸刻度、单位和色标布局。
- 结构示意图使用更醒目的相邻单元配色，去除遮挡图面的文字说明。

## 2.x

- 完成 PySide6 桌面 GUI 第一版：参数区、自动计算结果、结构图、二维切面、三维包络、u-v 方向图和扫描并集。
- 扫描输入统一为 x-z 切面扫描角和 y-z 切面扫描角，内部换算 u/v、theta/phi。
- 增加快速、标准、精细三种计算模式，支持 PNG、CSV、工程保存/加载和 Excel/WPS 验证工作簿。
- 修复扫描方向、固定切面、u-v 主瓣移动、扫描并集不是角度示意而是空间包络并集等关键验收问题。

## 1.x

- 基于 MATLAB 原型 `a.m` 复现核心算法逻辑，建立 Python 计算核心。
- 完成波长、阵列口径、远场边界、HPBW、经验最大扫描角、扫描损失、相邻单元相位差和总输入功率等自动推导结果。
- 建立后续桌面软件化所需的核心模块划分：几何、单元方向图、阵因子、功率密度、包络、扫描并集、导出。
